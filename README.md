# Museum De Pixel - Een Vergelijking tussen RDR2 & CS:GO

Dit is een websiteproject, 'Museum De Pixel', gemaakt door twee studenten van het Grafisch Lyceum Rotterdam. De site dient als een digitaal museum dat twee iconische games vergelijkt en naast elkaar plaatst: het cinematische **Red Dead Redemption 2** en het competitieve **Counter-Strike: Global Offensive**.

## Auteurs

Dit project is ontworpen en gebouwd door:

* **Max Paardekam:** Verantwoordelijk voor het visuele ontwerp en de Red Dead Redemption 2-sectie.
* **Floris Ebus:** Verantwoordelijk voor het ontwerp, de homepagina en de Counter-Strike-sectie.

## Projectinformatie

Dit project is ontwikkeld als opdracht voor de vakken 'Beroeps' en 'Proces' in het eerste jaar van de opleiding.

* **School:** Grafisch Lyceum Rotterdam
* **Opleiding:** Creative Software Developer (Jaar 1, Klas D1B)
* **Jaar:** 2025

## Gebruikte Technologieën

De website is gebouwd met standaard webtechnologieën:

* **HTML5:** Voor de structuur en semantiek van de pagina's.
* **CSS3:** Voor alle styling, layout (inclusief flexbox/grid voor de kaarten en split-screen), en het gebruik van CSS Variabelen voor thematisering (zoals te zien is in de RDR2-knoppen).
* **JavaScript:** Voor interactieve elementen, zoals de modal (popup) die in `index.html` is gedefinieerd.

## Structuur van het Project

De website is als volgt opgebouwd:

```

/
├── index.html          (De homepagina met de split-screen hero)
├── about.html          (De 'Over ons' pagina met info over de makers)
│
├── csgo/
│   ├── page1.html      (Een subpagina voor Counter-Strike)
│   ├── styles/
│   │   └── styles.css  (De centrale stylesheet voor het hele project)
│   ├── scripts/
│   │   └── script.js   (JavaScript voor de modal en andere interacties)
│   └── images/         (Afbeeldingen voor CS:GO en algemeen gebruik)
│
└── rdr2/
├── index.html      (Een subpagina voor Red Dead Redemption 2)
└── images/         (Afbeeldingen specifiek voor RDR2)

```

## Hoe te gebruiken

Om de website lokaal te bekijken:

1.  Zorg ervoor dat alle bestanden en mappen (met name `csgo` en `rdr2`) in dezelfde hoofdmap staan.
2.  Open het bestand `index.html` in een webbrowser.
